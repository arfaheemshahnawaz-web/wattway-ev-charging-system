# WattWay EV Charging System

Web-based EV charging station management system.

## Features

- Driver registration
- Charging station registration
- Admin dashboard
- Station availability updates
- Charging management

## Tech Stack

Frontend:
HTML  
CSS  
JavaScript  

Backend:
PHP  

Database:
MySQL  

Server:
XAMPP

## Installation

1. git clone https://github.com/arfaheemshahnawaz-web/wattway-ev-charging-system
2. Move project to XAMPP htdocs
3. Import database from `schema.sql`
4. Start Apache and MySQL
